# Upute za pokretanje projekta Tekme.ar

## Preduvjeti:
1. Instalirajte Python (verzija 3.7 ili novija).
2. Instalirajte Flask biblioteke pokretanjem `pip install flask`.

## Pokretanje projekta:
1. Pokrenite backend:
    - Otvorite terminal i idite u direktorij gdje se nalaze projektne datoteke.
    - Pokrenite `python app.py`.
    - Backend će se pokrenuti na `http://127.0.0.1:5000`.
2. Otvorite `index.html` u pregledniku.

## Funkcionalnosti:
- **Prikaz nadolazećih utakmica**: Podaci se dohvaćaju iz backenda.
- **Prognoze umjetne inteligencije**: Prikaz AI predviđanja.
